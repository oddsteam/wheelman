# Choosing Web Framework

Date: 2026-01-26

## Context

เรากำลังเริ่มทำ Wheelman app ใหม่ จากของเดิมที่ใช้ Go เป็น Backend และ React เป็น Frontend
ซึ่งทำไปเล็กน้อย แล้วไหน ๆ เราก็เริ่ม Fresh start ใหม่ แล้วก็อยากให้หลาย ๆ
คนสามารถเข้ามาช่วยพัฒนาได้ ไม่ว่าจะเป็นบทบาทไหนก็ตาม

## Decision

เลือกใช้ Ruby on Rails เพราะว่าทุกอย่างสามารถทำให้จบภายในตัว Framework เองได้
และก็คิดว่าเป็น Framework ที่ Friendly สำหรับทุก ๆ บทบาท ไม่จำเป็นต้องจบสายงานด้าน Computer
มาก่อน

## Consequences

มี Learning curve ในการศึกษา Framework สำหรับทุก ๆ คนที่ยังไม่เคยใช้งานมาก่อน
การเริ่มต้นเข้ามาร่วมพัฒนาอาจจะใช้เวลาทำความคุ้นเคยก่อนจะเริ่มงานได้จริง
